## Política de edição de ADRs

- ADR aceito **nunca é editado substancialmente**;
- correções factuais pequenas são permitidas e devem ser registradas
  (nota de rodapé com data);
- mudança de decisão gera **novo ADR**; o anterior recebe status
  `substituído por ADR-NNNN` — nunca é apagado.
